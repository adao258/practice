# 原型

1. 引用类型的四个规则

- 引用类型都具有对象的特性，可以自由扩展
- 引用类型都有一个隐式原型 prop 属性，属性值是一个普通对象
- 隐式原型 prop 的属性值都指向他的构造函数的显示原型 prototype 属性值
- 当试图得到一个对象的某个属性时，如果这个对象没有这个属性，那么他会尝试从他的隐式原型 prop
  里面(显示原型 prototype) 中去寻找
