# vue-comprehensive

## 项目搭建

0. 项目说明: vue-comprehensive 是用于练习各种 vue 组件而创建的 pc 版，包含了...
1. 创建项目

-   使用 vue-cli3 创建，vue 版本为 2.0

2. 安装依赖及插件

-   router: 路由管理
-   vuex: 状态管理
-   axios: 数据请求
-   sass/sass-loader: css 预处理
-
-   element-ui: 主要的组件库

3. 项目初始化过程

-   主题的布局采用圣杯布局,
-   问题 1: 导航栏的固定，
-   解决: 吸顶效果,最简单的处理 `position:sticky;` 但是兼容性不太好，也可以使用 relative 和 fixe。
