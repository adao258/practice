## @mixin

-   Mixins 可以包含任意内容且可以传递参数。通过@mixin 加名称的方式就可以定义一个 Mixins 模块，其中你可以添加任何你想重复使用的样式

```
@mixin button {
    font-size: 1em;
    padding: 0.5em 1.0em;
    text-decoration: none;
    color: #fff;
}
```

-   由于历史原因，连字符和下划线被认为是相同的，也就是说 @mixin button-large { } 和 @mixin button_large { } 是一样的。
