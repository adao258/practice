## 第三章 形状

-   **border-radius**创建椭圆并且自适应。它可以单独指定水平和垂直半径，只要用一个斜杠（/）分隔这两个值即可。设置宽高的一般可以成为椭圆，但这样做会有缺陷，就是改变元素就要调整相应的值，因此设置值时要设为百分比`border-radius: 50%;`
